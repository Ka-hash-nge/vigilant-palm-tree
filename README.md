# 阅读器 Windows 版

这是一个已经打包好的 Windows 桌面应用发布目录。

## 运行

在 Windows 上克隆仓库后，运行 `阅读器.exe`。

如果通过 Git 克隆，请先安装并启用 Git LFS：

```powershell
git lfs install
git lfs pull
```

## 参与

欢迎提交 issue、建议和 pull request。这个仓库目前保存的是可运行的 Windows 打包版本；如果要进行源代码级协作，建议后续再建立一个源代码仓库，保留构建脚本、源码和开发说明。

公开前已移除备份目录、source map、调试/构建中间文件，并扫描常见密钥格式与本机路径。请不要把个人配置、数据库、日志或账号凭据提交到仓库。
